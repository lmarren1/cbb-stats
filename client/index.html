Introducing Play Theory

A full-stack performance analytics platform built exclusively for Illini Men's Basketball

The Problem

Understanding the specific combination of players to pit against opponents is an age-old problem in basketball coaching.

Classic basketball analytics place too much emphasis on abstract boxscore-derived statistics to solve this issue. Such analyses lack the depth necessary to capture more than just the 'what' that occurs underneath fast-paced college basketball.

Inspired by modern approaches, I built Play Theory to uncover why certain lineups succeed against specific opponents.

Your New Analytics Solution

Championships are not built on box-scores. They're painstakingly fought for every posession by each player on the floor.

Play Theory breaks down games using publicly-available SideArm Sports play-by-play data and reconstructs it into actionable, interactive insights on lineup performance for coaches and players alike.

How It Works

Play Theory integrates:

  • An efficient PostgreSQL database
  • An asynchronous Node.js server
  • A Python analytics microservice
  • And customizable Plotly.js frontend visuals

all wrapped in a modern web interface built to deliver fast, yet in-depth analysis on team pace, player efficiency, and lineup synergy over the course of a game.
